# COGNIFYZ_TECHNOLOGIES
# Cognifyz Technologies – Python Development Internship Tasks

This repository contains the projects completed during my Python Development Internship at Cognifyz Technologies. The tasks were designed to strengthen problem-solving abilities, programming logic, object-oriented programming concepts, file handling, and data management skills using Python.

## Tasks Included

### Task 1: Python Knowledge Quiz Game

An interactive quiz application that tests Python programming knowledge through multiple-choice questions and score evaluation.

### Task 2: Number Pattern Generator

A menu-driven program that generates various numerical patterns such as triangles, pyramids, diamonds, and Pascal's Triangle using nested loops.

### Task 3: CRUD Task Manager

A task management system that allows users to create, view, update, and delete tasks while demonstrating CRUD operations and OOP concepts.

### Task 4: Temperature Converter

A utility application that performs temperature conversions between Celsius, Fahrenheit, and Kelvin using mathematical formulas.

### Task 5: Bookshelf Management System

A Python-based book browsing application that displays books, filters them by category, and presents details such as title, rating, price, and stock availability.

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* File Handling
* Data Structures
* API/Data Processing
* Exception Handling

## Learning Outcomes

Through these projects, I gained practical experience in Python programming, application development, data handling, user interaction, debugging, and software design principles.
